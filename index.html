What are the differences between `==` and `===`? Write an example for each case (if any)?

Phép == dùng để so sánh giá trị của 2 phần tử

Phép === dùng để so sánh kiểu và giá trị của 2 phần tử

VD: var a = 1; var b = '1'. Nếu dùng == thì kết quả là true vì nó chỉ so sánh 2 giá trị của 2 phần tử và cả 2 đều có cùng kết quả là 1. Nếu dùng === thì kết quả là false vì 2 trị thuộc 2 kiểu khác nhau, biến a có kiểu dữ liệu number còn biến b có kiểu dữ liệu là string.
